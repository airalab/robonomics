# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.x.x   | :white_check_mark: |
| 2.x.x   | :x:                |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

Please report vulnerability as github [issue](https://github.com/airalab/robonomics/issues/new) or directly to research@robonomics.network
